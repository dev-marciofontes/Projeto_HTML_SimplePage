O que é o HTML

(Voltar)

Você já se perguntou como são criadas as páginas da web que você visita todos os dias?
O HTML, sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem de programação responsável por estruturar e apresentar conteúdo na internet.
O HTML é a base fundamental para a criação de páginas web e funciona como uma espécie de esqueleto que dá forma e organização aos elementos que compõem um site.

As principais TAGs HTML

(Voltar)

As TAGs HTML são elementos utilizados para marcar e estruturar o conteúdo em uma página web. Existem várias TAGs disponíveis, cada uma com sua função específica. Vamos conhecer algumas das principais:

<html>: Essa TAG é usada para definir o início e o fim do documento HTML. Todo o conteúdo da página deve estar contido entre essas duas tags.

<head>: A TAG <head> é onde você define informações sobre o documento HTML, como o título da página, a codificação de caracteres e links para folhas de estilo.

<body>: A TAG <body> engloba todo o conteúdo visível da página, como textos, imagens e links. É dentro dessa TAG que a maior parte do trabalho é feita.

<h1>: a <h6>: Essas TAGs são usadas para definir os cabeçalhos do texto, do maior (h1) ao menor (h6). São úteis para destacar títulos e subtítulos.

<p>: A TAG <p> é usada para criar parágrafos de texto. É uma das TAGs mais comuns e amplamente utilizadas.

<a>: A TAG <a> é usada para criar links. Você pode definir o destino do link utilizando o atributo href.

Criando o seu Hello World

(Voltar)

Agora que você já conhece algumas das principais TAGs HTML, vamos criar o seu "Hello World", que é um exemplo básico de página web. Vamos lá!

  1. Abra um editor de texto simples, como o Bloco de Notas no Windows ou o TextEdit no macOS.
  2. Digite o seguinte código em texto normal: 
                        <!DOCTYPE html>
                    <html>
                        <head>
                            <title>Minha primeira página</title>
                        </head>
                        <body>
                            <h1>Olá, mundo!</h1>
                            <p>Esta é a minha primeira página web.</p>
                        </body>
                    </html>
  3. Salve o arquivo com a extensão ".html", por exemplo, "pagina.html". Certifique-se de salvar o arquivo com a codificação UTF-8.
  4. Abra o arquivo no seu navegador favorito.
  5. Parabéns! Você acaba de criar sua primeira página web simples usando o HTML.

Agora você pode personalizar e experimentar com diferentes TAGs HTML para adicionar mais conteúdo à sua página. Por exemplo, você pode adicionar imagens utilizando a TAG <img> ou criar uma lista usando as TAGs <ul> e <li>. Explore as possibilidades e divirta-se criando suas próprias páginas web!

Lembre-se de que esse exemplo é apenas um ponto de partida e o HTML oferece muitas outras opções e recursos para a criação de páginas web incríveis. Continue aprendendo e praticando para aprimorar suas habilidades em desenvolvimento web.
